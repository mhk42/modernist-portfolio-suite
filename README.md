# Corporate Modernist Portfolio Suite

**Live Site:** [View Live Project](https://mhk42.github.io/modernist-portfolio-suite/)  
**Presentation:** [View Project Presentation](https://mhk42.github.io/modernist-portfolio-suite/presentation/)

A unified ecosystem of premium web experiences built with a **Corporate Modernist** aesthetic. This project showcases the intersection of engineering precision and minimal design, leveraging Tailwind CSS for rapid, responsive deployment and visual consistency.

The suite is structured around four core pillars, all accessible through a central hub. The design philosophy prioritizes generous whitespace, high-contrast typography, and fluid responsiveness to create a refined, enterprise-grade user experience.

### 1. Engineering Index (`index.html`)

The central gateway of the project. This page functions as a master directory using a glassmorphism-inspired interface to organize and route visitors across the ecosystem. It establishes a premium first impression while maintaining clarity and structure.

### 2. Professional Portfolio (`portfolio.html`)

A personal branding platform designed for high-trust conversions.

- **Creator Archetype:** Positions the engineer as a forward-thinking builder and problem solver  
- **Reciprocity Integration:** Includes a live Calendly widget for frictionless consultation booking  
- **Narrative Structure:** Uses a modular grid to present experience and projects with logic-driven explanations and outcomes  

### 3. Client Webpage: NexusFlow AI (`client.html`)

A high-conversion SaaS landing page built for a fictional AI automation company.

- **Sales Funnel Architecture:** Guides users through a structured journey from lead magnet to conversion  
- **Persuasion Principles:** Implements scarcity through limited founding-member pricing and social proof via authority logos and performance metrics  
- **Responsive Navigation:** Features a custom-built mobile hamburger menu for seamless cross-device usability  

### 4. Design Style Guide (`design.html`)

The single source of truth for the entire suite. This document defines the core design tokens and reusable components used across all pages.

- **Color Palette:**  
  - CM-Dark `#1c1c1e`  
  - CM-Light `#f5f5f7`  
  - CM-Accent `#2f80ed`  
- **Typography:** Inter font family with consistent tracking and leading rules  
- **Components:** Standardized buttons, cards, and interactive UI elements
